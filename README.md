# Building Energy Audit System

A comprehensive energy efficiency assessment platform built on Stacks blockchain for tracking building consumption, conducting audits, and managing retrofit projects.

## Features

- **Building Registration**: Register and manage building profiles with ownership verification
- **Energy Audits**: Authorized auditors can conduct comprehensive energy assessments
- **Consumption Tracking**: Record and monitor electricity, gas, and water usage over time
- **Retrofit Project Management**: Plan, track, and measure energy improvement projects
- **Efficiency Analysis**: Calculate energy savings and improvements between periods
